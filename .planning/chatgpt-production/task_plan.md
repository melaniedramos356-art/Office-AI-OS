# ChatGPT App 成品制作模式

## 目标
将 Office-AI-OS 的 Word、PPT、Excel 入口改为生成统一创作任务单，由 ChatGPT App / Codex 使用网页搜索、技能和素材完成最终成品。

## 阶段

- [x] 明确职责边界与用户选择
- [x] 新增统一创作任务单生成器
- [x] 重写办公生成与改进入口，移除旧的本地成品生成链路
- [x] 调整桌面程序的结果呈现与保存方式
- [x] 编写并运行针对任务单的测试

## 约束

- 允许删除旧的本地成品生成代码，避免两套流程并存。
- 不把 ChatGPT 会员当作本地程序可调用的接口。
- 任务单必须包含内容、视觉、网页检索、制作和验收要求，且不重复描述。
