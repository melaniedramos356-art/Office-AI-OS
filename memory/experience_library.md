# Office-AI-OS 经验库

经验库分为两大块：程序迭代经验、制作技巧与素材查找经验。

## 程序迭代经验

- 新功能先做最小可运行版本，再逐步增强。
- 重复逻辑要及时抽成公共工具，避免死代码。
- 文件生成类功能不能覆盖用户原文件。
- 每个 Agent 都要有对应测试。

## 制作技巧与素材查找经验

- 版面设计先看信息层级，再看颜色和装饰。
- 文案生成先明确对象、目的和行动。
- 图片查找要记录来源和用途。
- 图片生成要写清主体、场景、风格、比例和用途。
- 国内活动视觉、营销海报、图标和模板优先查中国素材库。
- 中国素材库适合补充更贴近中文办公场景的审美和表达方式。
- 使用中国素材时同样要记录来源、关键词和用途。
- 优秀作品查找要同时看国内素材库和国际灵感库。

### 程序迭代经验：本轮迭代经验

- 本轮需求：请继续迭代系统，简化代码并升级制作技巧
- 程序迭代要保持小步提交，避免一次改动过大。
- 新增功能要保持代码合理性、安全性和简洁性。
- 重复逻辑要优先抽成公共工具，不保留死代码。

### 制作技巧与素材查找经验：本轮制作技巧经验

- 制作技巧要沉淀到 production_techniques 和 experience_library。
- 中国素材库适合补充国内海报、活动视觉、图标和模板参考。
- 优秀作品查找要同时看国内素材库和国际灵感库。
