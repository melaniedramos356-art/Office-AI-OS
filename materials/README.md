# 优秀素材库

把你认为优秀的办公素材放到对应板块的 `examples` 文件夹里。

如果素材同时适合 Word、PPT、Excel，或者你不确定应该放哪里，先放到：

```text
materials/shared/examples/
```

如果素材来自国内网站、中文办公案例、国内活动视觉或中文模板，优先放到：

```text
materials/china/examples/
```

当前支持文件类型：

```text
.md
.txt
.csv
```

运行学习：

```bash
python office_ai.py
```

示例输入：

```text
请学习素材库里的优秀素材
```
